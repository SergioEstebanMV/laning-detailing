# Eme Detailing - Página web

Proyecto desarrollado para la asignatura **FrontEnd** (Sumativa 1) de Ingeniería Informática en INACAP.

## 📌 Descripción
Sitio web promocional para un emprendimiento de lavado de autos a domicilio en Maipú. Presenta los servicios, precios y un formulario de contacto.

## 🛠️ Tecnologías utilizadas
- **HTML5 semántico** (header, main, footer, section, article, address)
- **CSS3** (variables, flexbox, media queries, hover/focus, transiciones)
- **Bootstrap 5** (sistema de grillas)
- **Google Fonts** (tipografía Bangers)
- ## ✨ Características
- Formulario de contacto que envía consultas directamente a WhatsApp
- Diseño responsive con media queries propias
- Efectos hover y focus para accesibilidad

## 🤖 Asistencia de IA
Utilicé **ChatGPT** como asistente para:
- Generar la estructura base de las tarjetas de servicios
- Optimizar las variables CSS y sugerir paleta de colores
- Mejorar la accesibilidad con contraste WCAG y estados focus
- Documentar el código con comentarios
